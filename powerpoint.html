Powerpoint


O que é o Microsoft Powerpoint

Microsoft PowerPoint é um programa utilizado para criação/edição e exibição de apresentações gráficas, originalmente escrito para o sistema operacional Windows e portado para a plataforma Mac OS X. A versão para Windows também funciona no Linux através da camada de compatibilidade Wine.



10 funções do PowerPoint que você talvez não conheça

1- Preencher texto com uma imagem.

2- Definir o mesmo tamanho para múltiplas imagens.

3- Criar molduras para imagens.

4- Inserir vídeos do YouTube na apresentação.

5- Habilitar o play automático em vídeos.

6-Gerar lorem ipsum em caixas de texto.

7- Criar botões clicáveis.

8- Combinar formas.

9- Dar zoom numa parte específica do slide

10- Salvar a apresentação em PDF



Em que o Microsoft powerpoint pode ajudar

Então, o PowerPoint permite que você use imagens, áudio e vídeo para ter um maior impacto visual. Essas dicas visuais e de áudio também podem ajudar o apresentador a ser mais improvisado e interativo com o público.
